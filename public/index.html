Welcome to the Flags API

The flags API provides information about all the countries in the world. This database can provide each country flag as a 1600px wide png image. In addition it can provide the country outline. Finally the api provides some trivia style questions for each country.

Example Images

Example Response Object

  
        { 
          name: 'United States',
          flag: './images/flags/united-states.png',
          outline: './images/outlines/united-states.png',
          questions: [ 'This country declared independence in 1776.' ] 
        }

/api/v1/country/

Endpoint will return all countries.

/api/v1/country/:id

Endpoint will return a specific country based on id.

/api/v1/facts

Endpoint will return all the trivia facts for all countries.

/api/v1/facts/:country_id

Endpoint will return the trivia facts for a single country.
