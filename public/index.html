World Flags API

The Flags API provides information about all the countries in the world. This database can provide each country flag as a 1600px wide image. In addition it can provide the country outline, and it provides some trivia style questions for each country.

Example Images

/api/v1/country/

Endpoint will return all countries.

/api/v1/country/:id

Endpoint will return a specific country based on id.

        {
          "id": 100,
          "name": "Switzerland",
          "flag": "./images/flags/switzerland.png",
          "country_outline": "./images/outlines/switzerland.png",
          "created_at": "2018-12-22T16:48:48.903Z",
          "updated_at": "2018-12-22T16:48:48.903Z"
        }

/api/v1/facts

Endpoint will return all the trivia facts for all countries.

/api/v1/facts/:country_id

Endpoint will return the trivia facts for a single country.

/signup

Endpoint will request a new user to sign up. Response upon successful sign up of new user:

        {
          "user": [
            {
              "id": 1,
              "username": "Name@example.com",
              "created_at": "2018-12-22T18:56:27.634Z",
              "token": "******EXAMPLE******"
            }
          ]
        }

/signin

Endpoint will request a new user to sign in. Response upon successful sign in of returning user:

        [
            {
                "id": 1,
                "username": "Name@example.com",
                "token": "******EXAMPLE******"
                "points": null,
                "created_at": "2018-12-22T18:56:27.634Z",
                "updated_at": "2018-12-22T18:56:27.634Z"
            }
        ]
